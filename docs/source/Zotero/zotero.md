# Zotero安装说明



## 建立软链接

因为软件大小的有限，我们将占据存储空间较大的storage文件夹独立出来，建立链接。

windows:

```
mklink /j F:\zotero\storage F:\nutcloud\我的坚果云\zotero\storage
```

linux:

```
ln -s /users/ouyang/dropbox/zotero/storage /users/ouyang/dev/zotero/storage
```
